# Exercices C# – Apprentissage et progression

## Description

Ce repository regroupe les exercices que je réalise dans le cadre de ma formation au Brevet fédéral d’informaticien (spécialisation développement).

L’objectif est de documenter ma progression en programmation, de pratiquer les concepts fondamentaux et de développer mes compétences en C# et en développement d’applications.

Je publie régulièrement de nouveaux exercices afin de consolider mes connaissances et d’améliorer ma logique de programmation.

---

## Objectifs de ce repository

- Pratiquer la programmation en C#
- Comprendre la programmation orientée objet (POO)
- Améliorer ma logique algorithmique
- Documenter ma progression
- Préparer mon futur métier de développeur

---

## Contenu

Ce repository contient différents types d’exercices :

- Exercices d’algorithmique
- Manipulation de tableaux

- Gestion de données
- Exercices SQL Server
